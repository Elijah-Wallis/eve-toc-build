# EVE OS: THE OWNER'S MANUAL

## 1. HOW TO START THE CAR
Open your terminal and run:
`python3 -m streamlit run eve_dashboard.py`

## 2. HOW TO REFUEL (If you change API Keys)
Open the fuel tank:
`nano ~/.zshrc`

## 3. HOW TO UPGRADE THE AUTOPILOT
If you want to change the rules (e.g., lower the safety threshold):
1. Edit the file: `knowledge_base/raw/eve_constitution.txt`
2. Re-ignite the memory: `python3 knowledge_base/ingest.py`

## 4. THE SARAH TEST
- If a screen is too complex for a 3rd grader, it is broken.
- Red = STOP. Green = GO.
